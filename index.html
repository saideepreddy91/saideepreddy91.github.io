Projects

Here is a list of projects I worked on and their brief description.


  • Loss mitigation problems using optimization algorithms
    •       During my internship in Altisource I researched and evaluated existing quantitative model implementations for loss mitigation problems (debt modifications) and proposed use of new algorithm(Genetic Algorithm) for improving performance
         Also evaluated algorithms such as Pattern Search and GA and used them in solving optimization problems; also analyzed the effects of constraints and initial point in these optimization algorithms.
  • Neuromorphic Engineering
    •       Simulated spiking neurons using different models in MATLAB and Artificial Neural Networks
         Worked on design of spiking neural networks and learning rules to train them.
  • Eigenfaces Vs Fisherfaces - Face Recognition
    •       Demonstrated the performance of eigenfaces and fisherfaces. The traditional eigenfaces algorithm uses PCA(Principal Component Analysis) for dimensionality reduction in face recognition.
         A new approach called fisherfaces is found to be insesitive to large variations in lightning and facial expressions. The error rates were found to be lesser in case of fisherfaces.
  • Finite Machine based controller
    •       Designed an optimal datapath(registers, functional units and point-to-point communication based steering logic) and implemented on FPGA using synthesizable code in VHDL.
  • Mathematical Function Plotter
    •      Designed a Mathematical Function Plotter and tried to implement it on FPGA using VHDL.
